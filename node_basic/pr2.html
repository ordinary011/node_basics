За допомогою fs створити папаку students В ній створити 2 папкм JS та Java В кожній з низ створити ще 2 папаки. 1800 та 2000 У кожній папці мають бути txt файли які х-ють кожного студента. Імя Фамілія Вік файлм має називатись відповідно до дати, коли цей файл був створений. Імя Фамілія та вік має бути одне з значень мисиву, names, surnames та ages відповідно. Наприклад let names = [Viktor, Vova, Dima] let surnames = [Pupkin, Frolov, Gagarin] let ages = [22,44,66] І файл виглядає так 23042019.txt -> Dima Gagarin 22 Потім за допомогою fs перенести студентів з JS до студентів Java, а студентів Java на місце студентів JS Потім за допомогою стрімів продублювати їх в папці backup
